UEditor-for-Zoomla!逐浪CMS基于.net4.0平台开发，集成了逐浪CMS的内容管理、商城模块、电子商务、社区、OA、项目管理、社会化办公、移动网站生成等应用，在原汁原味保证ueditor的各种功能时，还增加了word导入、CMS分段等功能，适合广大站长、开发者用来构建各种智能系统，高效敏捷。

**功能特色：**

 1. 支持word直接上传解析
 2. 支持原生UEditor编辑器的各项功能
 3. 支持自动生成逐浪CMS段落
 4. 支持代码编辑器
 5. 与逐浪CMS完美整合
